SpringBoot整合Redis

环境配置:

  • Linux
  • Redis 3.0.0（redis-3.0.0.tar.gz）

Linux配置

  • 安装gcc-c++

  • 解压redis.tar.gz （tar -zxvf redis-3.0.0.tar.gz）

进入解压目录：

  • cd redis-3.0.0
  • make // 用于编译

将Redis安装到指定目录

  • make PREFIX=/usr/local/redis install

启动Redis

  • ./redis-server
